# Proof of Broke Homepage

A static landing page for the Solana meme token concept **Proof of Broke ($BROKE)**.

## Files
- `index.html` — Main website
- `styles.css` — Responsive styling
- `script.js` — Copy contract-address interaction
- `proof-of-broke-logo.png` — Logo image

## Edit before launch
1. Replace `PASTE_CONTRACT_ADDRESS_HERE` in `index.html` with your real token contract address.
2. Replace social links in the `#join` section.
3. Update tokenomics if your final distribution changes.

## Suggested deploy
Upload this folder to Vercel, Netlify, Cloudflare Pages, or any static web host.

## Disclaimer
This website describes a meme token concept only. It does not provide financial advice.
